# Baile.js

Content choreography with CSS animations and JavaScript
## Usage



## Install

Standalone js:
Just include baile.min.js.
A global variable called ```Baile```
will be added.

From node:
```javascript
npm install baile-js
var Baile = require('baile-js')
var scene = Baile()
scene.select('.item').play('animation', '1s')
```
